# Identified Vulnerabilities

During the audit several vulnerabilities were analyzed.

## SQL Injection

SQL Injection occurs when user input is not properly sanitized before being used in database queries.

Impact:

- data leakage
- authentication bypass
- database manipulation

## Cross-Site Scripting (XSS)

XSS vulnerabilities allow attackers to inject malicious scripts into web pages viewed by other users.

Impact:

- session hijacking
- malicious redirects
- data theft
